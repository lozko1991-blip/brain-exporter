# 🧠 Brain API → XML Exporter

Автоматична вигрузка каталогу Brain.com.ua у формат XML (Rozetka / Prom.ua).  
Працює **повністю на GitHub** — без сервера, без Claude після налаштування.

---

## 📁 Файли в репозиторії

| Файл | Призначення |
|------|-------------|
| `export.py` | Python скрипт — качає Brain API, генерує XML |
| `config.json` | Твої налаштування: категорії, мова, назва магазину |
| `index.html` | Сайт для вибору категорій (GitHub Pages) |
| `.github/workflows/export.yml` | Розклад і налаштування автозапуску |
| `output/catalog.xml` | Готовий XML файл (створюється автоматично) |

---

## 🚀 Налаштування (один раз)

### Крок 1 — Форкнути або створити репозиторій

1. Завантаж всі файли в новий GitHub репозиторій
2. Назви його наприклад `brain-exporter`

---

### Крок 2 — Додати секрети (логін/пароль Brain)

1. Відкрий репозиторій на GitHub
2. Зайди: **Settings → Secrets and variables → Actions**
3. Натисни **"New repository secret"** і додай два секрети:

| Name | Value |
|------|-------|
| `BRAIN_LOGIN` | Твій email на opt.brain.com.ua |
| `BRAIN_PASSWORD` | Твій пароль (хешується MD5 автоматично) |

> ⚠️ Пароль зберігається зашифрованим, GitHub не показує його нікому

---

### Крок 3 — Вибрати категорії

**Варіант А — через сайт (зручно):**
1. Відкрий GitHub Pages: `https://ВАШ_АКАУНТ.github.io/brain-exporter/`
2. Постав галочки навпроти потрібних категорій
3. Натисни "Копіювати config.json"
4. Відкрий файл `config.json` в репозиторії → Edit → встав → Commit

**Варіант Б — вручну:**  
Відредагуй `config.json` прямо на GitHub:
```json
{
  "lang": "ua",
  "shop_name": "Назва твого магазину",
  "shop_url": "https://твій-сайт.ua",
  "output_file": "output/catalog.xml",
  "category_ids": [21, 155, 47]
}
```

---

### Крок 4 — Увімкнути GitHub Pages

1. **Settings → Pages**
2. Source: **Deploy from a branch**
3. Branch: **main**, folder: **/ (root)**
4. Save

Твій сайт: `https://ВАШ_АКАУНТ.github.io/brain-exporter/`

---

### Крок 5 — Перший запуск

1. Зайди: **Actions → Brain XML Export**
2. Натисни **"Run workflow"**
3. Вкажи ID категорій (або залиш порожнім — візьме з config.json)
4. Вибери мову: `ua` або `ru`
5. Натисни **"Run workflow"** (зелена кнопка)
6. Чекай 10-60 хвилин (залежить від кількості категорій)

---

## 🔗 Постійне посилання на XML

Після першого запуску файл збережеться в репозиторій.  
**Це посилання не зміниться ніколи:**

```
https://raw.githubusercontent.com/ВАШ_АКАУНТ/brain-exporter/main/output/catalog.xml
```

Вставляй його в:
- **Prom.ua** → Товари → Імпорт → YML посилання
- **Rozetka** → HUB → Товари → Імпорт → XML фід
- **Hotline**, **Google Shopping** — аналогічно

## 🔄 Автооновлення

XML оновлюється **автоматично кожні 6 годин**:
- 06:00 за Києвом
- 12:00 за Києвом  
- 18:00 за Києвом
- 00:00 за Києвом

Безкоштовний GitHub тариф дає **2000 хвилин Actions на місяць**.  
Одна вигрузка займає ~10-60 хв залежно від категорій.  
4 вигрузки на день × 30 днів × ~30 хв = ~3600 хв → може не вистачити для великих каталогів.

**Рішення для великих каталогів:** зміни розклад на раз на день:
```yaml
- cron: '0 3 * * *'   # тільки о 06:00 Київ
```

## 📊 Що є в XML файлі

- ✅ Назва товару (українська/російська)
- ✅ Ціна оптова (UAH)
- ✅ Ціна рекомендована роздрібна
- ✅ Всі фотографії (велике/середнє/мале)
- ✅ Виробник (vendor)
- ✅ Артикул
- ✅ Гарантія
- ✅ Країна виробник
- ✅ Наявність (кількість на всіх складах)
- ✅ Опис товару
- ✅ Всі характеристики (розмір, колір, матеріал тощо як `<param>`)
- ✅ Групування варіантів по `group_id` (розміри одного товару)
- ✅ Ієрархія категорій з батьками

## ❓ Часті питання

**Скільки часу займає вигрузка?**  
~1 хвилина на 500 товарів. 10 000 товарів ≈ 20 хвилин.

**Чи потрібен сервер?**  
Ні. Все працює на серверах GitHub безкоштовно.

**Чи потрібен Claude?**  
Тільки для первісного налаштування. Після — повністю автономно.

**Як підключитись до API Brain?**  
Напиши на support_b2b@brain.ua. Потрібна реєстрація на opt.brain.com.ua зі статусом Адміністратор.
